Fire.js

The litte DOM Helper for Javascript-Nerds,
who don't need big DOM-Frameworks to do stuff!

addClass, removeClass and hasClass

Fire.js adds functionality for adding and removing Classes of DOM Elements.
You also can check if an Element has a Class.
Check the code below to see how simple it is.

multiple Elements

Fire.js also works, if you want to select multiple Elements (for example: getElementsByTagName())

getElementsByClassName

This Framework adds a getElementsByClassName Function to DOM Elements, if this function does not exist (older browser like IE8,...).
