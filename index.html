Fire.js

Fire.js - The litte DOM Helper for Javascript-Nerds

Usage

Include Fire.js in your html-file -> Done.

<script src="your/path/fire.js"></script>

Examples

addClass, removeClass and hasClass

Fire.js adds functionality for adding and removing Classes of DOM Elements. You also can check if an Element has a Class. Check the code below to see how simple it is.

var elem = document.getElementById("elem1");
setTimeout(function(){
    if(elem.hasClass("active")) {
        elem.removeClass("active");
    } else {
        elem.addClass("active");
    }
},1000);

multiple Elements

Fire.js also works, if you want to select multiple Elements (for example: getElementsByTagName())

var elems = document.getElementsByTagName("div");
setTimeout(function(){
    if(elems.hasClass("active")) {
        elems.removeClass("active");
    } else {
        elems.addClass("active");
    }
},1000);

getElementsByClassName

This Framework adds a getElementsByClassName Function to DOM Elements, if this function does not exist (older browser like IE8,...).

var elems = document.getElementsByClassName("div");
setTimeout(function(){
    if(elems.hasClass("active")) {
        elems.removeClass("active");
    } else {
        elems.addClass("active");
    }
},1000);
